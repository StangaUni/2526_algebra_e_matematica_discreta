# Algebra e Matematica discreta — UniPD Informatica 2025/2026

Riassunti personali redatti a seguito della frequentazione del corso di **Algebra e Matematica discreta** del corso di laurea in Informatica presso l'Università degli Studi di Padova, anno accademico 2025/2026.

## Struttura della repository

```
.
├── Algebra/
│   └── Riassunti/          # Riassunti delle lezioni di Algebra
├── Matematica discreta/
│   └── Riassunti/          # Riassunti delle lezioni di Matematica discreta
└── Utils/
    ├── assets/             # Immagini e SVG di supporto
    └── *.md                # Schede riepilogative (es. preparazione compitini)
```

## Materiale non incluso — Lezioni

Le slide e il materiale delle lezioni **non sono stati caricati** in questa repository in quanto contrassegnati come materiale UniPD e soggetti a copyright.

Chi volesse aggiungerli localmente può farlo seguendo questi passi:

1. Creare una cartella `Lezioni/` nella root della repository
2. Rimuovere (o commentare) la riga `Lezioni/` dal file `.gitignore`
3. Aggiungere il materiale desiderato e fare push sul proprio account

> **Sconsiglio di lasciare le lezioni come materiale pubblico accessibile a chiunque**, sia per rispetto del copyright UniPD sia per tutelare il lavoro del docente.

## Visualizzazione

Tutti i file `.md` sono scritti e gestiti tramite **[Obsidian](https://obsidian.md)**. La repository include la configurazione `.obsidian/` per garantire la stessa esperienza visiva a chi utilizza lo stesso editor.

Non posso assicurare la medesima fruizione a chi utilizza visualizzatori diversi (GitHub, VS Code, ecc.): alcune formattazioni o collegamenti interni potrebbero non essere renderizzati correttamente.
